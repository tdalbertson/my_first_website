Creating Links, Absolute & Relative Links, and displaying an image

Today I learned about a few different things

  • Creating Links
  • Absolute & Relative Links
  • Displaying images on a webpage using HTML

Creating Links in HTML

To create a link in HTML we use the anchor tag <a>

For example:

About The Odin Project

Absolute & Relative Links

Absolute links always contain the protocol and domain of the destination

For example https://www.theodinproject.com

Relative links only include the file path to the other page, relative to the page you are creating the link on.

For example:

                
                    <body>
                        <h1>Homepage</h1>
                        <a href="pages/about.html">About</a> <--! Relative link here -->
                    <body>

Displaying images on a webpage using HTML

You can add images using the <img> tag along with its src

Don't forget to include the alt attribute to describe the image for screen readers!
